# Developer Guide

## Locally Development

### Installation

```sh
#
pnpm install
```

### Running

```bash
#
pnpm run dev
pnpm run dev:debug

#
pnpm run build

#
pnpm run start
pnpm run start:prod
```

### Testing

```bash
#
pnpm run test
pnpm run test:watch
pnpm run test:cov
pnpm run test:debug
pnpm run test:e2e
```
